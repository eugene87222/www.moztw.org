Firefox 新鮮事

  • 全新功能 Firefox Screenshots，在瀏覽器中拍照、存檔及分享畫面擷圖。
  • 全新功能 自動填寫表單中的地址資料（目前僅 en-US 版本提供）。
  • 全新功能 「選項」相關更新：
    • 新增搜尋框，方便快速找到項目。
    • 重新分類選項，更容易掃視各項設定。
    • 改善各選項的描述文字，協助瞭解不同選擇及實際套用的效果。
    • 改善資料收集相關選項，符合新版隱私權公告和資料收集策略。
  • 全新功能 於背景開啟的分頁將不會自動播放媒體內容（選擇該分頁後才會播放）。
  • 全新功能 改善 iOS 和 Android 的「將分頁傳送至裝置」功能，而且即使沒有 Firefox 帳號也能使用。
  • 變更 採用以 Rust 開發的 Encoding 標準實作取代原本的字元編碼轉換程式。
  • 變更 新增 AES-GCM 硬體加速。
  • 變更 升級 Safe Browsing 協定至第 4 版。
  • 變更 減少約 20% 安裝檔案大小。
  • 變更 改善下載檔案的安全性驗證。
  • 開發功能 檢測器新增 CSS「格線」面板。
  • 開發功能 更多開發者功能改進。

已知問題

  • 尚未修復 安裝 RelevantKnowledge 廣告軟體後可能無法正確啟動。在 Firefox Support 可以找到如何移除它的指引。
  • 尚未修復 使用 Windows 遠端桌面協定（RDP）執行的 Firefox 可能因為縮限的安全限制而無法播放聲音。在下個修正之前，可參考此問題的暫時性解法。
  • 尚未修復 Lenove IdeaPad 的「OneKey Theater」軟體可能造成 64 位元 Windows 7 的 Firefox 啟動失敗。重新安裝 32 位元版本的 Firefox 可以解決這個問題。
  • 尚未修復 Mac OS X High Sierra 的 bug 造成全螢幕模式有一些問題。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
