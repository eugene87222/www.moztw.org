Firefox 新鮮事

  • 全新功能 於 Windows 平台支援 WebVR，帶來身歴其境的網路體驗。請見 Mozilla VR 上實際運行的範例。
  • 全新功能 新增效能調校選項：
    • 啟用 Windows 10 周年更新版的 VP9 硬體加速，減少觀看影片時的電池消耗和 CPU 使用率。
    • 調整內容處理程序的數量，改進網頁載入和切換分頁的速度。
  • 全新功能 提供 Windows 版線上（stub）安裝檔，簡化安裝程序：
    • 在具備 2GB 以上 RAM 的 64 位元 Windows 系統，預設會安裝 64 位元版的 Firefox。
    • 仍可下載完整版的安裝檔。
  • 全新功能 網址列功能改進：
    • 直接在網址列以指定搜尋引擎搜尋。
    • 預設會顯示搜尋建議。
    • 輸入網域名稱時，Firefox 會優先使用 https 而非 http 連線。
  • 全新功能 更新書籤、歷史、已同步分頁等側邊欄，可將側邊欄移至右方。
  • 全新功能 WebRTC 支援立體聲麥克風。
  • 全新功能 在預覽列印中簡化列印內容。
  • 全新功能 OS X、macOS 可以在「系統偏好設定」設定快速鍵存取 Firefox 選單項目。
  • 全新功能 即使上次瀏覽的分頁數量很多，也能快速復原。
  • 全新功能 使用 screenshots 功能拍攝網頁擷圖，儲存於本地或上傳至雲端。此功能以 A/B 測試模式進行，並未開放給所有使用者。
  • 全新功能 新增白俄羅斯語（Belarusian, be）語系支援。
  • 修正 各種安全性修正。
  • 變更 Adobe Flash 外掛程式將預設為 click-to-play（點擊後播放），且僅允許在 http 和 https 網站使用。此變更並未對所有使用者生效，詳見 Firefox plugin roadmap。
  • 變更 雖然在過去版本可行，但 Firefox 並不支援版本降級。升級 Firefox 55+ 後，個人設定檔將無法以舊版 Firefox 開啟，未來的降級亦可能遇到問題。
  • 變更 改進程式更新介面，採用低侵擾性、更融入瀏覽器其他部分的設計。只有更新後 8 天都未重啟，以及關閉自動更新的使用者會看到這個變更。
  • 開發功能 未使用 SSL 的網站將無法存取 Geolocation API。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
